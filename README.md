## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites

```
node -v
npm -v
```

### Installing

1. Clone the repository

```
git clone https://github.com/rawda-developer/Mass-Streaming
cd Mass-Streaming
```

2. Backend

```
cd backend
npm install
npm run dev
```
